Welcome to OpenState SDN project.

To try OpenState feature you need to run our custom version of Mininet.

The following commands have been tested in a clean Mininet v2.1.0 VM. For download and help running Mininet please refer to http://mininet.org/

1) We need to update Mininet install scripts in order to download and configure OpenState switch/controller. Inside Mininet shell type the following commands:

cd mininet
git remote add openstate https://github.com/OpenState-SDN/mininet.git
git pull openstate master

2) We can now install OpenState switch/controller with the following options:

sudo ./util/install.sh -n3fy

The command above will run Mininet installer including OpenState softswitch and controller in only one shot!

3) All set here. Now you can check out some OpenState sample applications
